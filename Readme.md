# CRUD Películas FrontEnd II

## Descripción

Este proyecto es un CRUD (Crear, Leer, Actualizar, Eliminar) de películas, realizado en el contexto de un ejercicio de la materia Frontend II, con un enfoque en el uso de LocalStorage para almacenar la información de las películas en el navegador del usuario. El proyecto permite a los usuarios gestionar una lista de películas, realizar búsquedas, aplicar filtros y más.

## Funcionalidades

- **CRUD de películas**: Permite crear, leer, actualizar y eliminar entradas de películas.
- **Buscador de películas**: Implementación de un buscador que permite encontrar películas por nombre.
- **Filtro de películas**: Filtro para categorizar las películas, según género.
- **Almacenamiento local**: Se utiliza LocalStorage para guardar los datos de las películas, de forma que la información persista incluso después de recargar la página.

## Requisitos

- Navegador web moderno.

## Instrucciones de uso

1. **Clona el repositorio**: 
```bash
   git clone https://github.com/KennyFigueroa-UAO/CRUD_Peliculas_FrontEndII.git
```

2. **Instala las dependencias (si es necesario)**: 
   Si el proyecto usa alguna herramienta o librería, deberás instalar las dependencias. Por ejemplo, si usas npm:
```bash
   npm install
```

3. **Abre el archivo index.html en tu navegador**: 
   Navega al archivo `index.html` en tu proyecto y ábrelo en un navegador para interactuar con el CRUD de películas.

## Estructura del Proyecto

- `index.html`: Página principal con el formulario y la interfaz de usuario.
- `localStorage.js`: Lógica de JavaScript que maneja el CRUD de películas y la integración con LocalStorage.
- `style.css`: Estilos básicos para la presentación de la interfaz.

## Notas

La tarea se divide en tres partes:

1. **CRUD Películas**: Implementar el CRUD básico
2. **Buscador de películas**: Implementar la funcionalidad de búsqueda
3. **Filtro de películas**: Implementar el filtro para las películas
